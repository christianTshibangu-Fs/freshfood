THE FRESH FOOD PROJECT

Goal
This is an e-commerce application that helps sellers receive customer orders.
It consists of a user interface and an administrator interface :

    - The administrator can add, delete, or update new products and view all user orders.
    - Users can view their existing orders and place new ones.

Future features

The application will evolve to include features such as 
    - locating customers who place orders, 
    - sending notifications to the administrator or delivery driver, and enabling delivery of the order.
